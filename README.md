# -web-app
此项目为外卖web app
包括商家、商品、购物车、用户等多个子模块
技术栈为：Vue全家桶+es6+webpack等前端技术
采用模块化、组件化、工程化的模式开发


#技术选型：
前台数据处理/交互/组件化：
1、vue
2、vue-router
3、vuex
4、mint-ui
5、vue-lazyload、
6、vue-scroller、better-scroll、swiper（滑动库）
7、moment、data-fns(日期处理)

前后台交互：
mock数据:mockjs
接口测试:postman
ajax请求：axios

模块化：ES6、babel

项目构建/工程化：webpack、vue-cli、eslint

css预编译器:stylus
